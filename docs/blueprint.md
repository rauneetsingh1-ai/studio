# **App Name**: TeamUp Forge

## Core Features:

- User Profile Creation: Users create detailed profiles including skills, interests, bio, and project preferences.
- AI-Powered Matching: Use AI to analyze user profiles based on their semantic meaning (OpenAI embeddings). Then recommend the best teammate matches to the user. The tool determines if two profiles are a match.
- User Authentication: Secure user authentication using Firebase Authentication.
- Match Score Display: Display the match scores between users.
- Real-time Chat Rooms: Enable real-time communication between matched users using Firestore or Realtime Database.
- Secure Profile Data Storage: Store user profile data securely in Firestore.
- Real-time Team Collaboration: Provide tools and interfaces for real-time team collaboration on projects.

## Style Guidelines:

- Primary color: Soft violet (#B19CD9) to convey a sense of innovation and creativity.
- Background color: Light gray (#F0F0F0) for a clean and modern look.
- Accent color: Soft orange (#FFB347) to highlight key elements and call-to-action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined, neutral look.
- Use modern, outline-style icons to represent various skills and interests.
- Clean and intuitive layout with a focus on user profile clarity and easy navigation.
- Subtle transitions and animations to enhance user experience.
